# Contributing

Thank you for your interest in contributing! We welcome new automation recipes from the community.

## How to Submit a Recipe

1. **Fork** this repository
2. Create a new file in `/recipes/` following the naming convention: `your-recipe-name.md`
3. Use the [recipe template](templates/recipe-template.md) as a starting point
4. Write your recipe **in Polish** (this repo targets Polish SMBs)
5. Submit a **Pull Request** with a clear description

## Recipe Requirements

Every recipe must include:

- **Title** — clear, descriptive name
- **Problem** — what business problem it solves (1-2 paragraphs)
- **Solution** — step-by-step implementation guide with screenshots or diagrams where helpful
- **Tools Needed** — list of tools with links (prefer tools with free tiers)
- **Time Saved** — realistic weekly time savings estimate
- **Difficulty Level** — Beginner / Intermediate / Advanced
- **Tips** — common pitfalls and best practices

## Quality Guidelines

- Be specific — "click Settings → Integrations" is better than "go to settings"
- Include real-world examples relevant to Polish businesses
- Test your recipe before submitting
- No API keys, secrets, or sensitive data in examples
- Use Polish diacritics correctly (ą, ć, ę, ł, ń, ó, ś, ź, ż)

## Code of Conduct

Be respectful, constructive, and helpful. We're all here to save time and grow our businesses.

# Współpraca (wersja polska)

Dziękujemy za zainteresowanie! Zapraszamy do dodawania nowych przepisów na automatyzacje.

## Jak dodać przepis

1. **Sforkuj** to repozytorium
2. Utwórz nowy plik w `/recipes/` zgodnie z konwencją nazewnictwa: `nazwa-przepisu.md`
3. Użyj [szablonu przepisu](templates/recipe-template.md) jako punktu wyjścia
4. Napisz przepis **po polsku**
5. Wyślij **Pull Request** z jasnym opisem

## Wymagania

Każdy przepis musi zawierać: tytuł, problem, rozwiązanie krok po kroku, potrzebne narzędzia, szacowaną oszczędność czasu, poziom trudności i wskazówki.
